# Terrazas-Company 🏗️
### Sistema de Administración y Control de Servicios

Este es un sistema de escritorio desarrollado para la gestión integral de la empresa **Terrazas**, especializada en servicios de construcción y mantenimiento.

## 🚀 Funcionalidades Principales
* **Gestión de Clientes:** Registro y seguimiento de cartera de clientes.
* **Control de Inventario:** Administración de materiales y herramientas.
* **Manejo de Contratos:** Registro detallado de servicios y acuerdos.
* **Administración de Empleados:** Control de personal operativo.

## 🛠️ Tecnologías Utilizadas
* **Lenguaje:** Visual Basic .NET (.NET Framework / Core)
* **Base de Datos:** SQL Server
* **Interfaz:** Diseño moderno y minimalista en tonos azules y púrpuras.

## 📋 Requisitos para Ejecutar el Proyecto
Para que el proyecto funcione en tu computadora, asegúrate de tener:
1. **Visual Studio** (2019 o superior recomendado).
2. **SQL Server Management Studio** para la base de datos.
3. El archivo de solución `.sln` que se encuentra en la raíz.
